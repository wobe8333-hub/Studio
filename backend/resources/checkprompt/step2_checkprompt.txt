Step2 체크프롬프트 규칙

1. 파일 생성 확인
- script.txt 존재 및 길이 > 0
- sentences.txt 존재 및 문장 수 >= 3
- scenes.json 존재 및 씬 수 >= 1
- report.json 존재

2. UTF-8 인코딩 확인
- 모든 파일이 UTF-8로 저장되어야 함
- 한글 깨짐 문자(, ??) 포함 시 실패

3. 한글 내용 확인
- script.txt에 한글이 포함되어야 함
- sentences.txt에 한글이 포함되어야 함
- scenes.json의 narration에 한글이 포함되어야 함

4. 씬 구조 확인
- intro/body/conclusion 구조 존재
- 각 씬에 narration, visual_prompt, duration_sec 포함

5. Placeholder 검증
- "string" placeholder가 존재하면 실패
- 모든 narration, overlay_text, visual_prompt가 실제 값이어야 함

6. 문장 수 확인
- sentences.txt의 문장 수 >= 3 (권장: 20개 이상)
