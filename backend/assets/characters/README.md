# 캐릭터 이미지 폴더

이 폴더에는 그래픽 합성 렌더러에서 사용할 캐릭터 PNG 이미지를 저장합니다.

## 사용 방법

- 캐릭터 이미지는 PNG 형식이어야 합니다.
- 투명 배경(alpha channel)을 지원합니다.
- 권장 해상도: 1024x1024 이상
- 파일명 예시: `owner_default.png`, `character_01.png` 등

## 기본 캐릭터

기본 캐릭터가 없으면 시스템이 자동으로 기본값(`owner_default.png`)을 사용하려고 시도합니다.
캐릭터가 없어도 텍스트만으로 씬을 생성할 수 있습니다.
