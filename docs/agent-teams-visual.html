빠른 시작 — 4단계
01
팀 활성화 확인
Claude Code를 열고 에이전트 목록을 확인합니다. 9개가 표시되어야 합니다.
claude agents # 기대 출력: # backend-dev # frontend-dev # test-engineer # quality-reviewer # security-auditor # infra-ops # doc-keeper # ui-designer # ux-reviewer
02
미션 프리셋으로 팀 소환
AGENTS.md의 프리셋 문장을 Claude에게 그대로 입력합니다.
kas-feature 팀을 생성해줘. - backend-dev: API 로직 담당 - frontend-dev: UI 담당 - test-engineer: TDD 주도 - quality-reviewer: plan 모드 리뷰 작업: [요구사항 작성]
03
팀원과 직접 대화
Shift+Down으로 팀원을 순환하며 개별 지시를 줄 수 있습니다.
# in-process 모드 Shift+Down → 팀원 간 이동 Ctrl+T → 태스크 목록 열기 Enter → 팀원 세션 진입 Escape → 현재 작업 중단
04
자동 품질 게이트 확인
팀원이 태스크를 완료할 때마다 pytest + npm build가 자동 실행됩니다.
# TaskCompleted 시 자동 실행: 1. pytest tests/ -x -q --timeout=60 2. cd web && npm run build # 실패 시: 에이전트가 최대 3회 # 자가 치유 시도 → 실패 시 리드 보고
