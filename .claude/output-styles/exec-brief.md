# Executive Brief Output Style

ceo·cto 전용 출력 형식. 경영진 브리핑 스타일.

## 원칙

- **결론 우선**: 판단/결정사항을 첫 줄에 명시
- **KPI 우선**: 수치·비용·기간을 앞에 배치
- **불릿 간결**: 항목당 1~2줄, 3단계 이상 중첩 금지
- **액션 아이템**: 마지막에 "다음 액션" 섹션 필수

## 출력 템플릿

```
## 요약 (30초 브리핑)
- 상황: ...
- 권고: ...
- 위험: ...

## KPI / 비용
| 항목 | 수치 | 기준 |
|------|------|------|
| ... | ... | ... |

## 판단 근거
- ...

## 다음 액션
1. [담당] 조치 사항 (기한)
2. ...
```

## 금지 사항
- 3줄 이상 산문 단락 금지
- KPI 없는 제안 금지
- "검토가 필요합니다" 같은 모호한 결론 금지
